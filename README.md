# 患者信息查询系统

## 快速开始

要运行此应用，您需要PHP环境。以下是使用PHP内置服务器运行应用的步骤：

### 步骤1：打开命令行终端

### 步骤2：进入项目目录
```
cd d:\Download\webpage\html
```

### 步骤3：启动PHP内置服务器
```
php -S localhost:8000
```

### 步骤4：访问应用
打开浏览器，访问以下地址：
http://localhost:8000/patient_query.php

## 注意事项

- 确保您的PHP版本兼容此应用
- 数据库配置已在config.php文件中设置
- 如果遇到404错误，请检查PHP服务器是否正在运行，以及请求的文件是否存在
- 所有测试文件已清理，应用只包含核心功能文件

## 主要文件说明

- `patient_query.php`: 主应用文件，包含患者信息查询界面
- `patient_query_improved.js`: 改进的JavaScript功能，处理患者数据显示和交互
- `config.php`: 应用配置文件，包含数据库连接信息
- `api/`: 包含后端API文件

## 常见问题

**问题：** 访问页面时显示404错误
**解决方案：** 确保PHP服务器正在运行，并且请求的文件路径正确

**问题：** 无法获取患者数据
**解决方案：** 检查PHP服务器是否正在运行，以及相关API文件是否存在
